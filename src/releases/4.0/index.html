Bugzilla 4.0

Bugzilla 4.0 is the major release after Bugzilla 3.6. It contains major UI improvements, significant WebServices enhancements, automatic duplicate detection, AJAX user autocomplete, and a lot of other exciting new features.

  • First Release Candidate: November 2, 2010
