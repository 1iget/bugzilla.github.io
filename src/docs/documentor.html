Documenters' Guide

The Bugzilla documentation is part of the source code tree. It uses the reStructured Text (reST) format, as extended by our documentation compilation tool, Sphinx. To hack on it, install a copy of Bugzilla, read a bit about reST, and start making patches.

See here for the style guide.
