Bugzilla 3.0

Instead of releasing Bugzilla 2.24, the Bugzilla Project has released Bugzilla 3.0, thanks to the good progress we accomplished in the Bugzilla 3.0 Roadmap. In other words, development snapshots named 2.23.x have given birth to Bugzilla 3.0 (instead of 2.24).

Schedule:

  • Feature Freeze: October 30, 2006
  • Create release branch: February 26, 2007
  • Final Release: May 9, 2007
  • Download 3.0
  • New features in 3.0
  • Complete changelogs since prior versions
